# Amante Ideal - Frontend

Frontend del proyecto **Amante Ideal**, desarrollado con **React** y **Vite**, utilizando renderizado del lado del cliente (Client-Side Rendering).

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- Fetch API

## Descripción

Este frontend permite:

- Crear perfiles de posibles amantes
- Buscar amantes por interés
- Visualizar resultados en tiempo real
- Mostrar mensajes de éxito y error

La aplicación consume una API REST desarrollada en Node.js.
